# mengma-audio
双音轨合成工具

## 项目介绍

这是一个基于React和Capacitor构建的双音轨合成工具，支持Web和Android平台。

## 技术栈

- React + TypeScript
- Vite
- Tailwind CSS
- shadcn-ui
- Capacitor (跨平台支持)

## 开发指南

```sh
# 安装依赖
npm install

# 启动开发服务器
npm run dev

# 构建Web应用
npm run build

# 同步到Capacitor
npx cap sync
```

## Android构建

本项目配置了GitHub Actions自动构建，推送代码后会自动生成APK文件。
